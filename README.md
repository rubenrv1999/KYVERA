# 🚀 KYVERA - Agencia Digital

KYVERA es una landing page moderna desarrollada para ofrecer servicios de diseño y desarrollo web a empresas y emprendedores.

El proyecto está construido con HTML, CSS y JavaScript puro, incluyendo un formulario de contacto totalmente funcional mediante EmailJS.

---

## ✨ Características

- Diseño responsive
- Animaciones al hacer scroll
- Navbar fija
- Sección Hero
- Servicios
- Portfolio
- Formulario funcional con EmailJS
- Feedback visual durante el envío
- Diseño moderno y limpio

---

## 🛠 Tecnologías utilizadas

- HTML5
- CSS3
- JavaScript
- EmailJS
- Bootstrap Icons

---

## 📩 Funcionalidades

✔ Recepción de presupuestos mediante formulario.

✔ Envío de emails automático.

✔ Validación del formulario.

✔ Mensajes de éxito y error.
